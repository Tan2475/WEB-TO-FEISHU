# Web To Feishu (网页万能抓取到飞书)

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-green.svg)

一个强大的、高度可定制的 Chrome 浏览器扩展。无需编写代码，即可通过**可视化拾取**提取任意网页数据，并一键推送到**飞书多维表格 (Bitable)**、**飞书电子表格 (Sheets)** 或 **飞书知识库 (Wiki)**。

## ✨ 核心特性

- **🎯 零代码可视化拾取 (Visual Picker)**
  - 内置元素拾取器，鼠标点击即可自动生成精准的 CSS 选择器。
  - 完全不需要打开浏览器的 F12 开发者工具。
- **🕷️ 动态抓取规则引擎**
  - 可以为单个域名配置无限套并行抓取规则。
  - 支持提取文本 (text)、链接 (href)、图片 (src)、输入框值 (value)、HTML 源码 及 当前网页 URL。
- **🖱️ 自动化预交互 (Pre-action)**
  - 完美解决“悬停或点击才加载”的现代前端框架节点。扩展可自动执行点击/悬停并在延时后抓取数据。
- **📊 全兼容飞书双表生态**
  - **多维表格 (Bitable)**：自动组装结构化 JSON 追加任意记录。
  - **电子表格 (Sheets)**：智能读取首行作为表头映射，无需关心列序号，自动追加末尾行。
  - **知识库组件 (Wiki)**：自动调用 API 解析知识库底层的真实文件类型及 Token。
- **🔒 纯本地架构，确保隐私**
  - 没有第三方中间服务器。你的飞书 `App ID` 与 `App Secret` 仅仅保存在浏览器的本地存储 (`chrome.storage.local`)，扩展直接与飞书 Open API 通信。

## 🚀 安装指南

由于直接打包了 Tailwind 的编译输出，您可以直接以源码形式加载：

1. 克隆或下载本仓库代码到本地。
2. 打开 Chrome (或 Edge) 浏览器，进入扩展管理页面 `chrome://extensions/`。
3. 开启右上角的 **“开发者模式”**。
4. 点击 **“加载已解压的扩展程序”**，选择本项目的目录即可。

## 💡 使用教程

1. **配置飞书凭证：**
   - 点击扩展图标，然后在弹窗中点击右上角的 **“设置 ⚙️”** 进入选项页。
   - 填入您的飞书企业自建应用 `App ID` 和 `App Secret`（需开启相应的多维表格/电子表格/知识库的读写权限）。
   
2. **创建抓取规则：**
   - 在选项页点击 **“新增站点规则”**。
   - 输入 **“匹配 URL 关键字”**（例如 `xiaohongshu.com`）。
   - 在“提取字段”列表中，填入飞书的列名，并点击 **[ 🎯 拾取 ]** 按钮在网页上直接框选你想要的元素。

3. **一键提取数据：**
   - 浏览任意符合该关键字的网页。
   - 点击浏览器右上角的插件图标，在下拉框中选择你想使用的规则。
   - 输入或粘贴你想要推送的**目标飞书表格完整链接**。
   - 点击**提取数据**，在弹出预览框确认抓取数据无误后，点击**最后确认写入**！
